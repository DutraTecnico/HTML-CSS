Principais Formatações

Negrito / Destaque

Essa é uma forma de deixar a palavra em negrito, usando B (não semantica)

Agora temos outra forma de deixar a palavra em negrito, usando a tag STRONG (semantica)

Italico / Enfase

Essa é uma forma de deixar a palavra em Italico, usando I (não semantica)

Agora temos outra forma de deixar a palavra em Italico, usando a tag EM (semantica)

Texto Mrcado

Para marcar um texto usamos a tag MARK.

Texto Grande e Pequeno

agora teremos um texto grande e outro pequeno usando as tags BIG (grande (não semantico mais)) e SMALL(pequeno)

Texto Deletado

Podemos marcar um texto como excluido, para indicar que ele deve ser lido mas não deve ser considerado. Usamos a tag DEL

Sublinhado / Inserido

Vamos sublinhar um texto usando a tag U (não semantica)

Agora vamos subinhar usando a tag INS (semantica)

Sobrescrito

podemos inseris coisas do tipo x2 - 3, aqui sobrescrevemos o "2" usando a tag SUP

Subscrito

podemos inseris coisas do tipo H2O, aqui subscrevemos o "O" usando a tag SUB
