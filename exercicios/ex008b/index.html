Outras Formatações

Código Fonte / pre formatado

podemos facilitar a leitura de um codigo como document.getElementById("teste") para document.getElementById("teste"). Usando a tag CODE

podemos tambem usar a tag PRE para deixar uma formatação similar a de um codigo. Exemplo:(sem o PRE)
num = int(input("Digite um numero")) if num % 2 == 0; printf(f'O numero {num} é par) else: printf(f'O numero {num} é impar) print("Fim do programa")
(com o PRE)

    
        num = int(input("Digite um numero"))
            if num % 2 == 0;
                printf(f'O numero {num} é par)
            else:
                printf(f'O numero {num} é impar)
        print("Fim do programa")

Abreviações

Podemos abreviar usando a tag (ABBR) como por exemplo HTML E CSS. Se voce passar o mouser em cima ira aparecer o nome completo
